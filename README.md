**Mesto**

* Описание:
Mesto - это одностраничный сайт, иметирующий профиль.
Сайт оптимизирован для просмотра на различных устройствах.

**Реализация проекта**

* Flexbox
* Grid
* Медиа запросы
* Адаптивность
* БЭМ технология
* Формы

**Технологии**

* HTML
* CSS
